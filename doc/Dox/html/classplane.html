constructor for a plane

This constructor edits (a,b,c,d) as following - if d is non-zero (a',b',c',d') = (a/d,b/d,c/d,1) else if c is non-zero (a',b',c',0) = (a/c,b/c,1,0) else if b is non-zero (a',b',0,0) = (a/b,1,0,0) else (a'0,0,0) = (1,0,0,0)
